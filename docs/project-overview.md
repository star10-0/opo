# Project Overview

## ما هو النظام
نظام إدارة محطات وقود متعدد المحطات.

## المستخدمون
- المدير
- المحاسب
- العامل
- المشرف اختياري

## الكيانات التشغيلية الأساسية
- محطة
- يوم تشغيلي
- استلام مضخة
- قراءة عداد
- فترة سعر
- حساب عامل
- سيارة توزيع
- خزان
- صهريج وارد
- تسوية محاسبية
- سجل نشاط
- طلب موافقة

## التعاريف الأساسية
### المحطة
كيان مستقل يملك بياناته الخاصة.

### اليوم التشغيلي
اليوم الكامل للمحطة، يفتح تلقائيًا حسب الوقت المحدد من المدير.

### استلام المضخة
استلام فرعي داخل اليوم التشغيلي يرتبط بمضخة وعامل مسؤول.

### حساب العامل
الإغلاق الفردي للعامل ضمن استلامه داخل اليوم التشغيلي.

### سيارة التوزيع
وحدة بيع مستقلة عن المضخة لكن تدخل ضمن تقارير المحطة.

### الصهريج
عملية توريد واردة إلى الخزانات.

### الخزان
المخزون الفعلي للمادة داخل المحطة.
