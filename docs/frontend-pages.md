# Frontend Pages

## Dashboard
يعرض:
- مبيعات اليوم
- الفروقات المالية
- حالة الخزانات
- الصهاريج الأخيرة
- الحسابات المعلقة
- مبيعات سيارة التوزيع
- المقارنات اليومية والأسبوعية والشهرية

## Operational Day Page
يعرض:
- معلومات اليوم التشغيلي
- حالة اليوم
- قائمة المضخات
- قائمة الاستلامات الفرعية
- زر إغلاق اليوم

## Pump Assignments Page
يعرض:
- المضخة
- العامل المسؤول
- العمال المساعدين
- رقم البداية
- رقم النهاية
- الحالة
- زر تسجيل قراءة وسيطة
- زر إنهاء الحساب

## Worker Closing Page
يعرض:
- الكمية المباعة
- الإيراد
- المصاريف
- النقد المتوقع
- النقد الفعلي
- الفرق
- زر إرسال للمحاسب

## Deliveries Page
يعرض:
- إضافة صهريج
- بحث
- فلترة شهرية
- تعديل
- حذف منطقي
- ملاحظات

## Storage Tanks Page
يعرض:
- الخزانات
- الكميات الحالية
- السعة
- حد التنبيه
- تاريخ الحركات

## Distribution Vehicle Page
يعرض:
- السيارات
- جلسات العمل
- قراءة البداية
- قراءة النهاية
- المبيعات اليومية باسم السيارة

## Reports Page
يعرض:
- تقرير يومي
- أسبوعي
- شهري
- تقرير الفروقات
- تقرير سيارة التوزيع

## Permissions Page
خاصة بالمدير:
- المستخدمون
- الأدوار
- الصلاحيات
- المحطات المسموح بها

## Audit Log Page
يعرض:
- من قام بالعملية
- نوع العملية
- الوقت
- قبل
- بعد
- الموافقات
## First-run Onboarding
يعرض في أول تشغيل:
- حالة عدم وجود أي محطة بالنظام
- زر/نموذج "إنشاء أول محطة" حسب الصلاحيات
- اختيار المحطة النشطة وحفظها محليًا
- رسائل واضحة عند عدم وجود stationAccess للمستخدم
