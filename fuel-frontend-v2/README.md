# Fuel Frontend v2

واجهة React + Vite لنظام إدارة محطات الوقود.

## Scripts
- `npm run dev` تشغيل التطوير
- `npm run build` بناء إنتاجي
- `npm run check` فحص بناء إنتاجي
- `npm run preview` معاينة build على `0.0.0.0:4173`

## Environment
- `VITE_API_BASE_URL` عنوان API (مثال: `http://localhost:5000/api`)
- `VITE_API_TIMEOUT_MS` مهلة طلبات HTTP بالميلي ثانية
